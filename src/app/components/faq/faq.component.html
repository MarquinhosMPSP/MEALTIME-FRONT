Manual do restaurante

Para que você possa ter o máximo aproveitamento na utilização dos serviços da Mealtime, criamos um manual do restaurante. Abaixo você pode acompanhar um passo a passo com as principais dúvidas

Cadastro de itens do cardápio

Para realizar o cadastro de itens vá até o menu lateral e clique em "Cardápio". Na tela que será carregada você poderá acompanhar a lista de itens cadastrados, caso seja seu primeiro acesso essa lista estará vazia. Clique no botão de adicionar no canto superior esquerdo para adicionar itens. Na tabela cada item possui um botão de edição e exclusão dos itens

Cadastro de lugares do cardápio

Para realizar o cadastro de lugares vá até o menu lateral e clique em "Lugares". Na tela que será carregada você poderá acompanhar a lista de lugares cadastrados, caso seja seu primeiro acesso essa lista estará vazia. Clique no botão de adicionar no canto superior esquerdo para adicionar lugares. Na tabela cada mesa possui um botão de edição e exclusão dos itens

Área de Reservas

No menu lateral vá até o item "Reservas". Será aberta uma página com quatro abas. A primeira "Pendente" representa todas as reservas feitas por usuários que chegaram ao seu restaurante, ao lado de cada item existem dois botões o de aceitar e recusar, caso a reserva seja aceita ela ira para a aba "Aceitas" ou caso seja recusada para a aba "Canceladas". A aba "Finalizadas" representa todas as reservas que já foram fechadas para registro histórico. Em todas as abas é possível filtrar os resultados pela data em um filtro localizado no campo superio esquerdo

Área de pedidos

No menu lateral vá até o item "Pedidos". Será aberta uma página contendo todas as comandas abertas para o dia em questão. Cada item da tabela traz informações sobre a comanda, mesas, cliente e clicando sobre o item ele será expandido e mostrará todos os pedidos contidos nela

Área de Promoões

Para cadastrar uma nova promoção vá até o item "Promoções" no menu lateral, na tela que será aberta sera exibida uma lista com todos os itens cadastrados no cardápio do seu restaurante. Ao lado de cada item existe um botão de adicionar onde você poderá adicionar uma porcentagem de desconto a cada item

Área de Funcionários

No menu lateral clique no item "Funcionarios", na tela que será aberta existem todos os funcionários cadastrados no restaurante. Com esse cadastro seus funcionários poderão usar a aplicação mobile para criar comandas para clientes que não usaram o sistema ou mesmo reservar e fazer pedidos em uma mesa
